# Personal Dashboard - Chrome Extension
A custom Chrome "New Tab" override extension designed to provide a minimalist, aesthetically pleasing dashboard. This project was built to practice handling multiple asynchronous operations using Promises, Async/Await, and Geolocation APIs.

## 🚀 Features
- **Dynamic Backgrounds**: Fetches high-quality, nature-themed wallpapers from the Unsplash API on every new tab.

- **Cryptocurrency Tracker**: Real-time price tracking for Dogecoin (Market Price, 24h High, and 24h Low) using the CoinGecko API.

- **Real-time Clock**: A centered, large-format clock that updates every second.

- **Weather Integration**: Automatically detects your current location via the Geolocation API to display local weather conditions and temperatures from OpenWeatherMap.

- **Minimalist UI**: Clean, glassmorphism-inspired design with focused typography.

🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)

- **Tooling**: Vite (Build tool & Dev server)

- **Extension Framework**: Chrome Extension Manifest V3

- **APIs**:

  - Unsplash API (Photos)

  - CoinGecko API (Crypto)

  - OpenWeatherMap API (Weather via Scrimba proxy)

## 📂 Installation (Development Mode)
Since this extension is currently in development and not yet published to the public Chrome Web Store, you can install it locally:

### Clone the repository:
```
git clone https://github.com/yy933/Personal-Dashboard
cd personal-dashboard
```

### Install dependencies:
```
npm install
```

### Build the project:
```
npm run build
```

### Load into Chrome:

- Open Chrome and navigate to chrome://extensions/.

- Enable Developer mode (toggle in the top right).

- Click Load unpacked.

- Select the dist folder generated by the build process.

## 🧠 Key Learnings
This project served as a deep dive into:

- **Asynchronous JavaScript**: Implementing both .then() chains and async/await patterns to handle multiple network requests simultaneously.

- **Error Handling**: Implementing fallback UI states (e.g., default background images) when API limits are reached or requests fail.

- **Manifest V3**: Configuring chrome_url_overrides to personalize the browser experience.

- **Browser APIs**: Using navigator.geolocation to fetch coordinates and setInterval for DOM synchronization.

*Note: This project was created for educational purposes and is currently hosted locally as a personal productivity tool.*
